## Connecting to MongoDB

```javascript
const mongoose = require('mongoose');

mongoose.connect(process.env.MONGO_URI)
.then(() => {
    console.log("Database connected successfully");
})
.catch((err) => {
    console.log(err);
});
```

---

## Promise Concepts Learned

### Wrong

```javascript
.then(console.log("Connected"))
```

Problem:

```javascript
console.log()
```

runs immediately.

---

### Correct

```javascript
.then(() => {
    console.log("Connected");
})
```

Explanation:

Code executes only after successful connection.

---

## Major Issue Faced

Error:

```text
querySrv ECONNREFUSED
```

Reason:

Using MongoDB SRV connection string:

```text
mongodb+srv://
```

Node.js failed during SRV DNS lookup.

---

## Troubleshooting Performed

### Checked DNS

```bash
nslookup cluster0.tphn1ug.mongodb.net
```

### Checked SRV Records

```bash
nslookup -type=SRV _mongodb._tcp.cluster0.tphn1ug.mongodb.net
```

Result:

* DNS working.
* Atlas cluster reachable.

---

## Final Fix

Instead of:

```text
mongodb+srv://
```

Used:

```text
mongodb://
```

standard connection string from Atlas.

Result:

```text
Database connected successfully
```

## Interview Questions

### What is dotenv?

A package used to load environment variables from a .env file into process.env.

---

### Why use .env files?

To avoid hardcoding sensitive information such as:

* Database passwords
* API keys
* JWT secrets

---

### What is Mongoose?

An ODM (Object Data Modeling) library for MongoDB and Node.js.

Features:

* Schema validation
* Models
* Querying database

---

### What is process.env?

An object that contains environment variables available to the Node.js application.

---

### What is the difference between .then() and .catch()?

.then()

* Executes when promise succeeds.

.catch()

* Executes when promise fails.
