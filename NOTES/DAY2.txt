# Password Hashing

Installed:

```bash
npm install bcryptjs
```

Added:

```javascript
const bcrypt = require('bcryptjs');
```

Hashed password:

```javascript
const hashedPassword = await bcrypt.hash(password, 10);
```

Stored:

```text
$2b$10$...
```

instead of:

```text
123456
```

Concept Learned:

Passwords should never be stored in plain text.

---

# bcrypt Concepts

Hashing:

```javascript
bcrypt.hash(password, 10);
```

Verification:

```javascript
bcrypt.compare(password, hashedPassword);
```

Salt Rounds:

```javascript
10
```

Meaning:

Higher value = More secure but slower.

A route defines how an application responds to a specific HTTP request.


## What is a Controller?

A controller contains business logic executed when a route is called.
